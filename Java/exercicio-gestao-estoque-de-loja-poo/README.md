\# Exercício: Controle de Estoque (POO)



Este projeto simula um sistema de gerenciamento de estoque para produtos, desenvolvido como parte do curso de Java Completo do Prof. Nélio Alves.



\## 📋 Funcionalidades

\- Cadastro de produto com nome, código e preço.

\- Entrada de estoque (Compra).

\- Saída de estoque (Venda) com validação para evitar saldo negativo.

\- Atualização dinâmica de preços.

\- Exibição de informações formatadas do produto.



\## 🚀 Conceitos Aplicados

\- \*\*Encapsulamento\*\*: Uso de atributos `private` e métodos `get/set` para proteção de dados.

\- \*\*Construtores\*\*: Inicialização obrigatória dos dados essenciais do produto.

\- \*\*Lógica de Negócio\*\*: Validação de saldo em estoque antes de concretizar vendas.

\- \*\*Interatividade\*\*: Menu de opções utilizando loops `do-while` e tratamento de buffer do `Scanner`.



\## 🛠️ Tecnologias

\- Java 17 (ou superior)

\- Eclipse IDE / STS
